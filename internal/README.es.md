# Carpeta `internal`

Esta carpeta contiene **herramientas y utilidades internas** construidas para el equipo de desarrollo (por ejemplo: CLIs, scripts de migración empaquetados, evaluadores de prompts, etc.).

A diferencia de los scripts simples en `/scripts`, estas son herramientas estructuradas con sus propias dependencias, tests y configuraciones de paquete.

- **Propósito principal**: centralizar utilidades robustas utilizadas internamente por el equipo de ingeniería.
- **Recomendación**: documenta cómo instalar y usar cada herramienta interna.
